The Position Properties

The position property specifies the type of positioning method used for anelement (static, relative, absolute, fixed, or sticky).Each value affects the element’s location, layout, and behavior differently. You can use the top, right, bottom, and left properties to specify the distance of the element from the edge of the viewport
